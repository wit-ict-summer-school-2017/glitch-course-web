#Show Sponsors

Currently the Spnosor table is incomplete - it only lists the sponsor names, and not the club being sponsored:

~~~html
  <table class="ui table">
    <thead>
      <tr>
        <th>Sponsor</th>
        <th></th>
        <th></th>
      </tr>
    </thead>
    <tbody>
	    #{list items:sponsors, as:'sponsor'}
	      <tr>
	        <td>${sponsor.name}</td>
            <td></td>
	        <td></td>
	      </tr>
	    #{/list}
    </tbody>
  </table>
~~~

We can fix this now:

~~~html
  <table class="ui table">
    <thead>
      <tr>
        <th>Sponsor</th>
        <th>Clubs</th>
        <th></th>
      </tr>
    </thead>
    <tbody>
	    #{list items:sponsors, as:'sponsor'}
	      <tr>
	        <td>${sponsor.name}</td>
          <td> 
            <table class "ui table">
              <tr>
               #{list items:sponsor.support, as:'club'}
                 <td>${club.name}</td> </tr>
               #{/list}
               </tr>
             </table>            
          </td>
	        <td></td>
	      </tr>
	    #{/list}
    </tbody>
  </table>
~~~

Run the app now to see if this page renders as expected.
